Columns

A container can be divided into multiple columns. This can be used to create multi-column pages.

Our containers combine the container/row/column grid-based layout tools often seen in other popular CSS frameworks.

Note that this will not work with a full-width container. (Because a full-width container has no defined size overall, I don't see a good way for us to define the ideal size of each column for responsive purposes.)

Each column

When using container columns, you have the option of using the ca-column tag to define each column. But it's not a requirement. Any element will work.

Span

The container supports up to six columns. When using more than three or four, you'll probably want some columns to be bigger than others. That's where span comes into play.
